## Solarfog.vim

Vim colorscheme based on solarized with minimal syntax highlighting

### Screenshot

![solarfog.vim](./screenshot.png)

### Installation

    Plug 'VSngg/solarfog.vim'
    
### Usage

    set termguicolors
    colorscheme solarfog
